# ☀️ Co‑Vacances

Co‑Vacances est une plateforme de mise en relation entre familles monoparentales.

L'objectif est de permettre à des parents solos de :

- réduire le coût de leurs vacances
- partager un logement
- créer du lien social
- offrir des compagnons de vacances aux enfants

## Fonctionnalités

- Création de profil
- Recherche de co-vacancières
- Publication d'annonces
- Messagerie privée
- Système de compatibilité
